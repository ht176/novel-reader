## ✅ 功能列表

### 1. Calibre 集成 🔌
**优先级**: P0 | **预计工作量**: 2-3 天

#### 功能描述
- 连接 Calibre Content Server
- 浏览 Calibre 图书库
- 从 Calibre 导入书籍到本地书架
- 支持书籍元数据同步

#### 技术实现
- [ ] 创建 `src/services/calibre.ts`
- [ ] 实现 Calibre API 客户端（搜索、获取详情、下载）
- [ ] 添加 Calibre 配置页面（服务器地址、端口）
- [ ] 在书架页面添加"从 Calibre 导入"入口
- [ ] 支持批量导入

#### API 参考
```
GET http://<calibre-server>:8083/api/v1/search?query=<keyword>
GET http://<calibre-server>:8083/api/v1/book/<id>
GET http://<calibre-server>:8083/api/v1/book/<id>/cover
GET http://<calibre-server>:8083/get/<id>/format/EPUB
```

#### 验收标准
- [ ] 可配置 Calibre 服务器地址
- [ ] 可搜索 Calibre 图书库
- [ ] 可预览书籍详情（封面、简介、作者）
- [ ] 可下载书籍到本地书架
- [ ] 支持 EPUB/MOBI/AZW3 格式

---

### 2. 听书功能 (TTS) 🎵
**优先级**: P0 | **预计工作量**: 3-4 天

#### 功能描述
- 文本转语音朗读
- 支持多语音引擎（浏览器原生 + 云端 API）
- 播放控制（播放/暂停/停止/语速/音调）
- 后台播放支持

#### 技术实现
- [ ] 创建 `src/services/tts.ts`
- [ ] 实现 Web Speech API 封装
- [ ] 可选：集成 ElevenLabs/Edge TTS
- [ ] 添加阅读器 TTS 控制 UI
- [ ] 支持后台播放（Service Worker）
- [ ] 音频进度与文本同步高亮

#### 验收标准
- [ ] 可选择朗读语音（中文/英文）
- [ ] 可调节语速（0.5x - 2.0x）
- [ ] 可调节音调
- [ ] 支持后台播放
- [ ] 朗读时自动滚动文本
- [ ] 支持定时关闭（15/30/60 分钟）

### 4. 数据导出/导入 📤
**优先级**: P1 | **预计工作量**: 1-2 天

#### 功能描述
- 导出所有数据（书籍、进度、设置、书源）
- 导入备份数据
- 支持增量备份
- 支持导出为通用格式（JSON/OPDS）

#### 技术实现
- [ ] 创建 `src/services/backup.ts`
- [ ] 实现全量导出（IndexedDB → JSON）
- [ ] 实现导入解析（JSON → IndexedDB）
- [ ] 添加备份管理页面
- [ ] 支持自动备份（每天/每周）
- [ ] 支持导出为 OPDS 格式（兼容其他阅读器）

#### 验收标准
- [ ] 可导出全部数据为 JSON 文件
- [ ] 可导入 JSON 备份文件
- [ ] 导入时有数据冲突处理（覆盖/跳过/合并）
- [ ] 支持自动备份到本地
- [ ] 可选：支持备份到云存储（WebDAV/Calibre）
