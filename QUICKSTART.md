# 🚀 快速启动指南

## 第一次使用

### 1. 安装依赖

```bash
cd novel-reader
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173

### 3. 导入示例书源（可选）

打开浏览器控制台 (F12)，复制粘贴并运行：

```bash
node scripts/import-example-sources.js
```

或者手动添加书源：
1. 进入"书源管理"页面
2. 点击"+ 添加书源"
3. 配置书源信息

## 功能使用指南

### 📚 导入本地书籍

1. 点击书架页面的"📥 导入"按钮
2. 选择 TXT 或 EPUB 文件
3. 等待解析完成
4. 点击书籍开始阅读

### 🔍 搜索在线书籍

1. 点击"🔍 书城"按钮
2. 输入搜索关键词
3. 选择书源（可选）
4. 点击搜索结果
5. 点击"📥 加入书架"

### ⚙️ 配置书源

1. 进入"书源管理"页面
2. 点击"+ 添加书源"
3. 填写书源信息：
   - 书源名称
   - 基础 URL
   - 搜索 URL 模板（使用 `{keyword}` 占位符）
   - HTML 选择器配置
4. 点击"测试"验证书源
5. 启用书源

### 📖 阅读设置

1. 进入"设置"页面
2. 配置默认字体大小
3. 选择默认主题（日间/夜间/护眼）
4. 调整默认行间距

### 💾 数据备份

**导出数据：**
1. 进入"设置"页面
2. 点击"📤 导出数据"
3. 保存 JSON 备份文件

**导入数据：**
1. 进入"设置"页面
2. 点击"📥 导入数据"
3. 选择备份文件
4. 确认导入

## 常见问题

### Q: 书源搜索失败？
A: 检查以下几点：
1. 书源是否已启用
2. 书源 URL 是否正确
3. HTML 选择器配置是否正确
4. 网站是否有反爬限制

### Q: 导入 EPUB 失败？
A: 目前 EPUB 解析功能较为基础，建议：
1. 使用标准 EPUB 格式
2. 避免加密的 EPUB 文件
3. 优先使用 TXT 格式

### Q: 数据会丢失吗？
A: 数据存储在浏览器 IndexedDB 中：
- 清除浏览器缓存会删除数据
- 建议定期导出数据备份
- 不同浏览器数据不互通

### Q: 支持哪些格式？
A: 
- ✅ TXT（智能分章）
- ✅ EPUB（基础支持）
- ⏳ MOBI（计划中）
- ⏳ UMD（计划中）

## 技术栈

- **框架**: Vue 3.4 + TypeScript
- **构建**: Vite 5
- **状态管理**: Pinia
- **路由**: Vue Router
- **数据库**: Dexie.js (IndexedDB)
- **HTML 解析**: Cheerio
- **移动端**: Capacitor (可选)
