## ⚠️ 已知问题

### 1. EPUB 导入解析问题 ✅ 已修复
**问题描述**: 模拟模式生成的 EPUB 文件在导入时解析失败

**错误信息**:
```
[Parser] EPUB 解析失败：Can't find end of central directory : is this a zip file ?
```

**原因分析**: 
- JSZip 在浏览器环境中的导入方式需要特殊处理

**解决方案**: 
- 修复为 `const JSZip = await import('jszip').then(m => m.default);`
- 使用 JSZip 生成完整的 EPUB 文件结构

**当前状态**: ✅ 已修复并验证（2026-03-28 20:00）

### 2. 真实 Calibre 服务器连接
**问题描述**: 无法连接到真实的 Calibre 服务器

**原因**: 
- NAS (192.168.1.8) 未在当前网络
- 或 Calibre Content Server 未运行

**解决方案**: 
- 使用模拟模式进行前端功能测试
- 连接到家庭 Wi-Fi 后测试真实服务器
